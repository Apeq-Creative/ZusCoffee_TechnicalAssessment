# Saucedemo Playwright Tests

End-to-end tests for [saucedemo.com](https://www.saucedemo.com) written with Playwright + TypeScript.

## Prerequisites

- Node.js ≥ 18

## Setup & run

> **Zipping this project?** Exclude `node_modules/` — it is large, platform-specific, and regenerated by `npm ci`.

```bash
npm ci
npx playwright install chromium
npm test
```

That's it. The single `npm test` command installs nothing extra and runs all tests headlessly.

## What is tested

| Test | Description |
|------|-------------|
| Login — happy path | Logs in with valid credentials and asserts the inventory page loads |
| Login — invalid password | Submits wrong credentials and asserts the error message is shown |
| Purchase flow | Dynamically finds the two cheapest items, adds them to the cart, completes checkout, and verifies the order-confirmation page |

The two cheapest items are determined at runtime by reading every item's price from the DOM and sorting — no hard-coded product names.

## Optional commands

```bash
npm run test:headed    # run with a visible browser
npm run test:report    # open the HTML report after a run
```
